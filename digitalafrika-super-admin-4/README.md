# DigitalAfrika SuperAdmin

## Run Locally

**Prerequisites:** Node.js + PostgreSQL

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env` and adjust PostgreSQL credentials
3. Start API backend: `npm run server`
4. Start frontend: `npm run dev`
